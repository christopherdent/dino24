## Specs as of 2/10/24

- Ruby version 3.2.2

- Rails version 7.1.3

## Steps to run API locally

- npm install

- bundle install

- rails db:migrate

- rails db:seed

- Visit localhost:3000/api/v1/eras to confirm database seeded and then fire up the [React Frontend](https://christopherdent.github.io/dinofinder-frontend) to connect with it.
